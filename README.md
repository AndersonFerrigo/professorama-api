#professorama-api

Professorama API.

Desenvolvida para realizar a comunicação entre o Aplicativo Professorama e o banco de dados MySQL.

Desenvolvida na linguagem Java 8 utiliza os frameworks como Spring Boot, JPA e Hibernate.
